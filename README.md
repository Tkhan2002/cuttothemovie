# CutToTheMovie ✂️🎬

The honest movie time site. Every theater lists fake showtimes — we add 25 minutes so you know when the movie **actually** starts.

## 🌐 Live Site
👉 https://tkhan2002.github.io/cuttothemovie

## What it does
- Shows real start times for AMC, Regal, and Cinemark
- Click any showtime to buy tickets directly
- User accounts powered by Firebase

## Built with
- HTML, CSS, JavaScript
- Firebase Authentication
